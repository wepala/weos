# How-to Guides

How-to guides are **task-oriented** — they assume you already have WeOS running and know the basics. Each guide solves a specific problem.

| Guide | Task |
|-------|------|
| [Install WeOS]({% link _howto/install-weos.md %}) | Install from source, binary release, or Docker |
| [Configure the Database]({% link _howto/configure-database.md %}) | Switch between SQLite and PostgreSQL |
| [Create a Resource Type]({% link _howto/create-resource-type.md %}) | Define new content types via presets or CLI |
| [Manage Resources]({% link _howto/manage-resources.md %}) | Create, list, update, and delete content via CLI and API |
| [Deploy to Production]({% link _howto/deploy-to-production.md %}) | Deploy with Docker to Cloud Run or any container host |
| [Add an HTML Template]({% link _howto/add-html-template.md %}) | Create templates with `data-weos-*` attributes |
| [Manage Persons and Organizations]({% link _howto/manage-persons-organizations.md %}) | CRUD operations for people and organizations |
| [Seed Development Data]({% link _howto/seed-data.md %}) | Populate your local database with test data |
